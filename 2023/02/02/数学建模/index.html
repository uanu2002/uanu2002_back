整数规划

整数规划(IP)问题：数学规划中部分或全部变量限制为整数。

松弛变量和剩余变量：将不等式约束转化为等式约束，例如$x_1+x_2\ge10$转化为$x_1+x_2+x_3=10,x_3\ge0$，$x_3$称为松弛变量。相减时称为剩余变量。

  • 纯整数规划：所有决策变量均为整数。
  • 全整数规划：所有决策变量、系数和常数均为整数。
  • 混合整数规划：部分决策变量为整数。
  • 0-1整数规划：所有决策变量只能取0或1。

割平面法

  1. 先添加松弛变量和剩余变量，将原约束方程组中的不等式约束转换为等式约束。

  2. 线性规划求解，若出现基变量（决策变量）$x_i$不是整数，其对应的约束条件为$x_i+\sum_{j\in J}x_jb_{ij}=b_{i0}$，其中J是非基变量下标集合。

  3. 令$Z(b)=floor(b)为b的整数部分；S(b)=b-floor(b)为b的小数部分$。则原式变换为$x_i+\sum_{j\in J}x_j(Z(b_{ij})+S(b_{ij}))=Z(b_{i0})+S(b_{i0})$。

  4. 将整数和小数部分分别移至等式两侧：$x_i+\sum_{j\in J}x_iZ(b_{ij})-Z(b_{i0})=S(b_{i0})-\sum_{j\in J}x_jS(b_{ij})$。左侧为整数，右侧为$S(b_{i0})$（小于0）减去一个正数，进而获得一个新的约束条件：$x_i+\sum_{j\in J}x_iZ(b_{ij})-Z(b_{i0})\le 0$，再添加剩余变量转换成等式约束。

匈牙利算法

主要用于01规划问题。

互斥约束问题：需要根据不同情况调整约束条件
$$
y=1时，约束条件为:4x_1+5x_2\le200\
y=0时，约束条件为:3x_1+5x_2\le180\
转化为不等式约束：\
4x_1+5x_2\le200+(1-y)M\
3x_1+5x_2\le180+yM\
$$
此时只需要$M$取无穷大即可。

互斥约束推广：
$$
从p个约束条件：\sum^n_{j=1} a_{ij}x_j\le b_i(i=1,2,\cdots,p)中恰好选择q个约束条件\
y_i=0,选择第i个约束条件\
y_i=1,不选第i个约束条件\
转化为不等式约束:\
\sum^n_{j=1}a_{ij}x_j\le b_i+y_iM\
\sum^p_{i=1}y_i=p-q
$$
$y_i$使得第$i$项失效。

指派问题

标准形式：n个人完成n项工作，每个人只能完成一项工作，每个工作只能由一个人完成，求最优分配使得（时间、成本）总代价最小。
$$
C=(c_{ij}){n\times n}为指派问题的系数矩阵，c{ij}为第i个人完成第j项工作的代价\
X=(x_{ij}){x\times n}为指派问题的解矩阵，x{ij}=1为第i个人做第j项工作，0为不做\
s.t.
\sum^n_{j=1}x_{ij}=1(i=1,\cdots,n)\
\sum^n_{i=1}x_{ij}=1(j=1,\cdots,n)\
x_{ij}=0或1(i,j=1,\cdots,n)
$$
非标准形式的指派问题：

  • 最大化指派问题

    令代价矩阵$C$中最大元素为m，转化为求最小化问题$B=(b_{ij}){n\times n}=(m-c{ij})_{n\times n}$

  • 人数和工作数不等

    人少：添加虚拟人，其代价均为0

    工作少：添加虚拟工作，其代价均为0

  • 一个人可做多项工作

    添加多个相同的人

  • 某工作一定不能由某人做

    令该人做该工作的代价无穷大

指派问题的匈牙利解法的一般步骤：

  1. 变换指派问题的系数矩阵$(c_{ij})$为$(b_{ij})$，使得各行各列中都出现0元素：
    1. 将$(c_{ij})$的每行元素都减去该行的最小元素
    2. 再从所得新矩阵的每列元素中减去该列的最小元素
  2. 进行试指派，找最优解：在$(b_{ij})$中找尽可能多的独立0元素，若能找出n个独立0元素，则该n个预算对应解矩阵的元素为1，其余为0，即为最优解。找0元素步骤：
    1. 从只有一个0元素的行（列）开始，给0元素加圈，删去其所在行（列）的其他0元素。
    2. 给只有一个0元素的列（行）中的0元素加圈，删去其所在行（列）的0元素。
    3. 反复进行1、2步骤，直到无法找到满足要求的元素。
    4. 若仍有没有加圈的0元素，则该行（列）至少有两个0元素。则从含0元素最少的行（列）开始，比较该行（列）各0元素所在列（行）中0元素的数目，选择0元素少的列加圈，然后删去其行和列上的其他0元素。重复操作直到所有未删除的0元素都被加圈。
    5. 若加圈0元素数目m等于n，则为最优解，若$m\le n$，则进行下一步。
  3. 作最少的直线覆盖所有0元素：
    1. 对没有加圈的行打:heavy_check_mark:。
    2. 对已经打:heavy_check_mark:的行中所有含删去0元素的列打:heavy_check_mark:。
    3. 再对已经打:heavy_check_mark:的列中含加圈0元素的行打:heavy_check_mark:。
    4. 重复2、3步骤，直到无法继续。
    5. 对没有打:heavy_check_mark:的行画横线，有打:heavy_check_mark:的列画纵线，得到覆盖所有0元素的最少直线数$l$。若$l\le m$，则回到2.4重新选择指派。若$l=m\lt n$，则进行下一步。
  4. 变换矩阵$(b_{ij})$增加0元素：在没有被直线覆盖的所有元素中找到最小元素，然后打:heavy_check_mark:各行都减去该最小元素，打:heavy_check_mark:各列都加上该最小元素（保证系数矩阵不出现负元素），回到2重新开始。

非线性规划

一般形式：
$$
minf(x)\
s.t.\
h_j(x)\le 0\quad j=1,2,\cdots,q\
g_i(x)=0\quad i=1,2,\cdots,p\
$$
eg.
$$
minf(x)=x_{1}^2+x_{2}^{2}+x_{3}^{2}+8\
s.t.\left{
\begin{array}
xx_1^2-x_2+x_3^2 \ge 0 \
x_1+x_2^2+x_3^2 \le 20 \
-x_1-x_2^2+2=0 \
x_2+2x_3^2=3 \
x_1,x_2,x_3 \ge 0
\end{array}
\right.
$$
线性规划中01规划使用不等式约束$0\le x \le 1$

非线性规划中01规划使用非线性等式约束$x(1-x)=0$

最佳投资方案：投资额最小而总收益最大，即总收益/投资额最大。

import numpy as np
from scipy import optimize

# 定义目标函数
def objective(x):
    return x[0] ** 2 + x[1]**2 + x[2]**2 +8
 
# 定义约束条件
def constraint1(x):
    return x[0] ** 2 - x[1] + x[2]**2  # 不等约束
 
def constraint2(x):
    return -(x[0] + x[1]**2 + x[2]**2-20)  # 不等约束
 
def constraint3(x):
    return -x[0] - x[1]**2 + 2        # 等式约束
 
def constraint4(x):
    return x[1] + 2*x[2]**2 -3           # 等式约束

# 定义约束条件
con1 = {'type': 'ineq', 'fun': constraint1}
con2 = {'type': 'ineq', 'fun': constraint2}
con3 = {'type': 'eq', 'fun': constraint3}
con4 = {'type': 'eq', 'fun': constraint4}
 
# 4个约束条件
cons = ([con1, con2, con3,con4])  
 
# 决策变量的符号约束
b = (0.0, None) # 即决策变量的取值范围为大于等于0
bnds = (b, b ,b) # 所有决策变量范围

# 求解
x0=np.array([0, 0, 0]) # 定义决策变量初始值
solution = optimize.minimize(objective, x0, method='SLSQP', \
                    bounds=bnds, constraints=cons) # method为求解的算法

非线性规划初值不同可能会影响结果：将计算得到的最优解作为初值再次计算，直到得到结果变差。

二次规划

目标函数为二次函数，约束条件为线性。

一般形式：
$$
min\frac{1}{2}x^THx+f^Tx\
s.t.\
Ax\le b\
Aeq\cdot x=beq\
lb\le x\le ub
$$

import numpy as np
import cvxopt


def quadprog(H, f, L=None, k=None, Aeq=None, beq=None, lb=None, ub=None):
# L即A，K即b
    n_var = H.shape[1]

    P = cvxopt.matrix(H, tc='d')
    q = cvxopt.matrix(f, tc='d')

    if L is not None or k is not None:
        assert(k is not None and L is not None)
        if lb is not None:
            L = np.vstack([L, -np.eye(n_var)])
            k = np.vstack([k, -lb])

        if ub is not None:
            L = np.vstack([L, np.eye(n_var)])
            k = np.vstack([k, ub])

        L = cvxopt.matrix(L, tc='d')
        k = cvxopt.matrix(k, tc='d')

    if Aeq is not None or beq is not None:
        assert(Aeq is not None and beq is not None)
        Aeq = cvxopt.matrix(Aeq, tc='d')
        beq = cvxopt.matrix(beq, tc='d')

    sol = cvxopt.solvers.qp(P, q, L, k, Aeq, beq)

    return np.array(sol['x'])

多目标规划

一般形式：
$$
min(f_1(x),f_2(x),\cdots,f_p(x))\
s.t.\
g_i(x)\le 0\quad (i=1,2,\cdots,m)\
h_j(x)=0\quad (j=1,2,\cdots,l)
$$
正、负偏差变量：
$$
令d_i^0表示目标函数z_i的目标值(i=1,2,\cdots,l)\
正偏差变量d_i^+=max(f_i-d_i^0,0)表示决策值超过目标值的部分\
负偏差变量d_i^-=-min(f_i-d_i^0,0)表示决策值未达到目标值的部分\
恒有d_i^+\times d_i^-=0，d_i^+,d_i^-非负且通常d_i^-\le d_i^+.
$$
绝对约束：线性规划中的等式约束和不等式约束。

柔性目标约束：对绝对约束添加正、负偏差变量。

（1）第i个目标要求恰好达到目标值，即正、负偏差变量都要尽可能地小，这时$Min\quad d_i^-+d_i^+$

（2）第i个目标要求不超过目标值，即允许达不到目标值，就是正偏差变量要尽可能地小，这时$Min\quad d_i^+$

（3）第i个目标要求超过目标值，即超过量不限，但必须是负偏差变量要尽可能地小，这时 $Min\quad d_i^-$

多目标规划的一般数学模型

加权系数法

给每个目标函数一个权重，将多目标转化为单目标。
$$
min_{x\in D}Z(x)=\sum^r_{i=1}w_iZ_i(x)
$$

理想点法

利用单目标规划对每个目标单独求解，得到每个目标$z_i(x)$对应的理想点$d_i$，再构建新的目标函数$min\quad \varphi[(z(x))]=\sqrt{\sum_i[z_i(x)-d_i]^2}$，转化为单目标。

最大最小法

采取保守策略，在最坏的情况下，寻求最好的结果。
$$
min_{x\in D}max_{1\le i\le r}Z_i(x)
$$

约束条件法

给除某个目标外的所有目标函数添加约束条件（范围）$z_i^{min}\le z_i\le z_i^{max}$，进而将目标函数转化为不等式约束，变为单目标。

连贯法

根据目标优先级，依次将单个目标作为单目标规划求解，求解结果作为等式约束加入。

评价方法分类：

  • 主观赋权法：综合指数法、层次分析法等
  • 客观赋权法：主成分分析法、理想解法（TOPSIS）等

层次分析法

层次分析法（AHP）：解决多目标的复杂问题的定性与定量相结合的决策分析方法。

典型应用：

  • 用于最佳方法的选取
  • 用于评价类问题
  • 用于指标体系的优选

步骤：

  1. 建立层次结构
  2. 构造判断（成对比较）矩阵
  3. 层次单排序及其一致性检验
  4. 层次总排序及其一致性检验

1.建立层次结构模型：将决策的目标、因素和决策对象按关系分层。

最高层：决策的目的、要解决的问题。

最低层：决策时的备选方案。

中间层：考虑的因素、决策的准则。

（相邻的两层，高层为目标层，低层为因素层）

2.构造判断矩阵：将因素两两比较，采用相对尺度（元素数量不宜超过9个）。

若矩阵满足$a_{ij}\cdot a_{jk}=a_{ik}$，则称为一致阵（秩为1，唯一非零特征根为n）。

一致阵非零特征根n所对应的特征向量归一化后可作为权重（权向量）。

若不为一致阵，则需要进行一致性检验，进行下一步。

3.层次单排序及其一致性检验

不一致阵的最大特征根$\lambda\ge n$。

一致性指标（定性）：$CI=\frac{\lambda-n}{n-1}$，CI越接近0，一致性越好。

随机一致性指标RI，为确定值。

一致性比率（定量）：$CR=\frac{CI}{RI}$，当CR小于0.1时，认为不一致程度在容许范围内。

当通过一致性检验后，该特征根对应的特征向量归一化后可作为权重。

一致阵特征向量简化计算：对矩阵先进行列向量归一化，再求行和后归一化，得到的向量可近似为权重。

4.层次总排序及其一致性检验

当完成对每层计算对上层中每个元素的权重后，计算最低层相对于最高层的权重得分，进行排序。

层次总排序的一致性比率：$CR=\frac{a_1CI_1+a_2CI_2+\cdots+a_mCI_m}{a_1RI_1+a_2RI_2+\cdots+a_mRI_m}$，其中$CI_i$为下层对于上层第$i$个因素的权重向量计算得到的一致性指标，当CR小于0.1时，通过一致性检验。

import numpy as np
import pandas as pd
import warnings


class AHP:
    def __init__(self, criteria, b):
        self.RI = (0, 0, 0.58, 0.9, 1.12, 1.24, 1.32, 1.41, 1.45, 1.49)
        self.criteria = criteria
        self.b = b
        self.num_criteria = criteria.shape[0]
        self.num_project = b[0].shape[0]

    def cal_weights(self, input_matrix):
        input_matrix = np.array(input_matrix)
        n, n1 = input_matrix.shape
        assert n == n1, '不是一个方阵'
        for i in range(n):
            for j in range(n):
                if np.abs(input_matrix[i, j] * input_matrix[j, i] - 1) > 1e-7:
                    raise ValueError('不是反互对称矩阵')

        eigenvalues, eigenvectors = np.linalg.eig(input_matrix) # 返回特征值及其特征向量的集合

        max_idx = np.argmax(eigenvalues)
        max_eigen = eigenvalues[max_idx].real # .real获取实数部分
        eigen = eigenvectors[:, max_idx].real
        eigen = eigen / eigen.sum()

        if n > 9:
            CR = None
            warnings.warn('无法判断一致性')
        else:
            CI = (max_eigen - n) / (n - 1)
            CR = CI / self.RI[n]
        return max_eigen, CR, eigen

    def run(self):
        max_eigen, CR, criteria_eigen = self.cal_weights(self.criteria)
        print('准则层：最大特征值{:<5f},CR={:<5f},检验{}通过'.format(max_eigen, CR, '' if CR < 0.1 else '不'))
        print('准则层权重={}\n'.format(criteria_eigen))

        max_eigen_list, CR_list, eigen_list = [], [], []
        for i in self.b:
            max_eigen, CR, eigen = self.cal_weights(i)
            max_eigen_list.append(max_eigen)

理想解（TOPSIS）法

决策矩阵：m个备选方案，每个方案有n个属性。

一般步骤：

  1. 向量规范化求规范决策矩阵：决策矩阵$A=(a_{ij}){m\times n}$，规范化决策矩阵$B=(b{ij}){m\times n},b{ij}=a_{ij}/\sqrt{\sum^m_{i=1}a_{ij}^2},i=1,2,\cdots,m,\quad j=1,2,\cdots,n$（调整量纲）

  2. 构成加权规范阵：设决策人给定各属性的权重向量为$w=[w_1,w_w,\cdots,w_n]^T$，则加权规范阵$C=(c_{ij}){m\times n},c{ij}=w_j\cdot b_{ij},i=1,2,\cdots,m\quad j=1,2,\cdots,n$

  3. 确定正理想解$C^*$和负理想解$C^0$：正理想解的效益型属性取最大值，成本型属性取最小值，负理想解则相反。

  4. 计算各方案到正、负理想解的距离：
    $$
    方案d_i到正理想解的距离：\
    s_i^*=\sqrt{\sum^n_{j=1}(c_{ij}-c_j^*)^2},i=1,2,\cdots,m\
    方案d_i到负理想解的距离：\
    s_i^0=\sqrt{\sum^n_{j=1}(c_{ij}-c_j^0)^2},i=1,2,\cdots,m\
    $$

  5. 计算各方案的排队指标值（即综合评价指数）：$f_i^*=s_i^0/(s_i^0+s_i^*),i=1,2,\cdots,m$

  6. 按$f_i^*$由大到小排列方案的优劣次序

数据预处理（在步骤1前进行）方法：同时将成本型变为效益型

  • 线性变换：
    $$
    效益型属性：b_{ij}=a_{ij}/a_j^{max}\
    成本型属性：b_{ij}=1-a_{ij}/a_j^{max}
    $$

  • 标准0-1变换：
    $$
    效益型属性：b_{ij}=\frac{a_{ij}-a_j^{min}}{a_j^{max}-a_j^{min}}\
    成本型属性：b_{ij}=\frac{a_j^{max}-a_{ij}}{a_j^{max}-a_j^{min}}
    $$

  • 区间型属性的变换（即最优值不是最值，而是在某个区间内）：
    $$
    设最优属性区间为[a_j^0,a_j^*]，上下限为[a_j^{‘},a_j^{‘’}]\
    b_{ij}=
    \begin{cases}
    1-(a_j^0-a_{ij})/(a_j^0-a_j^{‘}),若a_j^{‘}\le a_{ij}\le a_j^0 \
    1,若a_j^0\le a_{ij}\le a_j^\
    1-(a_{ij}-a_j^    )/(a_j^{‘’}-a_j^*),若a_j^*\le a_{ij}\le a_j^{‘’} \
    0,其他
    \end{cases}
    $$

  • 向量规范化：同步骤1。（不能将成本型转换成效益型）

  • 标准化处理：（不能将成本型转换成效益型）
    $$
    b_{ij}=\frac{a_{ij}-\bar{a_j}}{s_j},i=1,2,\cdots,m,\quad j=1,2,\cdots,n\
    其中\bar{a_j}=\frac{1}{m}\sum^m_{i=1}a_{ij},s_j=\sqrt{\frac{1}{m}\sum^m_{i=1}(a_{ij}-\bar{a_j})^2},j=1,2,\cdots,n
    $$

聚类：非监督学习，不依赖标签。

目标：最大化类内的相似性，最小化类间的相似性。

分类：

  • Q型：对样品分类。
  • R型：对变量（指标）分类，主要用于降维。

距离：样品间的相似度量。

  • 欧氏距离：$d(x_i,x_j)=[\sum^p_{k=1}(x_{ik}-x_{jk})^2]^{1/2}$
  • 绝对距离：$d(x_i,x_j)=\sum^p_{k=1}|x_{ik}-x_{jk}|$
  • 明氏距离：$d(x_i,x_j)=[\sum^p_{k=1}|x_{ik}-x_{jk}|^m]^{1/m}$
  • 切氏距离：$d(x_i,x_j)=max_{1\le k\le p}|x_{ik}-x_{jk}|$
  • 方差加权距离（标准化的欧氏距离）：$d(x_i,x_j)=[\sum^p_{k=1}(x_{ik}-x_{jk})^2/s_k^2]^{1/2}$
  • 马氏距离：$d(x_i,x_j)=\sqrt{(x_i-x_j)^T\sum^{-1}(x_i-x_j)},其中\sum^{-1}为向量x和y的协方差矩阵的逆矩阵$
  • 兰氏距离：$d(x_i,x_j)=\frac{1}{p}\sum^p_{k=1}\frac{|x_{ik}-x_{jk}|}{x_{ik}+x_{jk}}$
  • 杰氏距离：$d(x_i,x_j)=[\sum^p_{k=1}(\sqrt{x_{ik}}-\sqrt{x_{jk}})^2]^{1/2}$

K-平均聚类算法

将n个数据聚成k类。

步骤：

  1. 从n个数据对象中任选k个对象作为初始聚类中心。
  2. 将每个聚类对象的均值作为聚类中心，计算每个对象与这些中心的距离，并根据最小距离对对象重新进行划分。
  3. 根据新的聚类划分重新计算均值。
  4. 重复步骤3、4直到每个聚类不再发生变化。

特点：

  • 只适用于聚类均值有意义的场合。
  • 必须指定k的个数。
  • 对噪声和孤立点数据敏感。

灰色预测

白色系统：系统的内部特征是完全已知的。

黑色系统：系统的内部信息对外界一无所知。

灰色系统，系统的内部信息部分已知，部分未知，

灰色预测法：对含有不确定因素的系统预测的方法。用等时距观测到的一系列数量值构造灰色预测模型，预测未来某一时刻的特征量，或达到某一特征量的时间。

类型：

  • 灰色时间序列预测：预测未来某一时刻的特征量，或达到某一特征量的时间。
  • 畸变预测：预测异常值出现的时刻或时区。
  • 系统预测：对系统行为特征指标建立一组相互关联的灰色预测模型，预测系统中众多变量间的互相协调关系的变化。
  • 拓扑预测：将原始数据做曲线，寻找定值出现的所有时刻，并构建数列，预测该定值发生的时刻。

灰色生成数列：基于原始数据生成新的数据，从而弱化其随机性，显现其规律性。

生成方法：设原始数列为$x^{(0)}=(x^{(0)}(1),x^{(0)}(2),\cdots,x^{(0)}(n))$

  • 累加生成：
    $$
    1次累加生成数列:\
    x^{(1)}(k)=\sum^k_{i=1}x^{(0)}(i),k=1,2,\cdots,n\
    \cdots\
    r次累加生成数列:\
    x^{(r)}(k)=\sum^k_{i=1}x^{(r-1)}(i),k=1,2,\cdots,n,r\ge 1
    $$
    即每次结果为前一次数列的累加。

  • 累减生成：
    $$
    x^{(1)}为x^{(0)}的1次累减生成数列:\
    x^{(1)}(k)=x^{(0)}(k)-x^{(0)}(k-1),k=2,3,\cdots,n
    $$
    即每次结果为前一次前后相邻两个数据相减。

  • 加权邻值生成：
    $$
    z^{(0)}为x^{(0)}在权\alpha下的邻值生成数列:\
    z^{(0)}(k)=\alpha x^{(0)}(k)+(1-\alpha)x^{(0)}(k-1),k=2,3,\cdots,n\
    特别地，当\alpha = 0.5时,称z^{(0)}为均值生成数
    $$

设$x^{(0)}$为原始数列，其1次累加生成数列为$x^{(1)}$，$x^{(1)}$的邻值生成数列为$z^{(1)}$。则$x^{(1)}$的灰导数为$d(k)=x^{(0)}=x^{(1)}(k)-x^{(1)}(k-1)$，GM(Grey Model)(1,1)的灰微分方程为：$d(k)+az^{(1)}(k)=b或x^{(0)}(k)+az^{(1)}(k)=b$，a称为发展系数，b称为灰作用量，$z^{(1)}(k)$称为白化背景值。

则微分方程可以表示为$Y=Bu$，化简可计算

原灰微分方程对应白微分方程为$\frac{dx^{(1)}(t)}{dt}+ax^{(1)}(t)=b$，

GM（1,1）灰色预测步骤：

  1. 数据检验与处理：
    计算数列级比：$\lambda(k)=\frac{x^{(0)}(k-1)}{x^{(0)}(k)},k=2,3,\cdots,n$，若级比都落在可容覆盖区间$X=(e^{\frac{-2}{n+1}},e^{\frac{2}{n+1}})$内，则可以使用GM（1,1）模型进行预测。如果不满足，则进行变化处理，如平移变化$y^{(0)}(k)=x^{(0)}(k)+c,k=1,2,\cdots,n$，使得满足要求。预测时要再逆变换回。

  2. 、建立GM（1,1）模型：$x^{(0)}(k)+az^{(1)}(k)=b$

    计算a，b的值，获得其白微分方程：$\frac{dx^{(1)}(t)}{dt}+ax^{(1)}(t)=b$

    微分方程解为$x^{(1)}(t)=(x^{(0)}(1)-\frac{b}{a})e^{-a(t-1)}+\frac{b}{a}$

    则预测值$x^{(1)}(k+1)=(x^{(0)}(1)-\frac{b}{a})e^{-a(k)}+\frac{b}{a},k=1,2,\cdots,n-1$

    进而得到原始数据预测值$\hat{x}^{(0)}(k+1)=\hat{x}^{(1)}(k+1)-\hat{x}^{(1)}(k),k=1,2,\cdots,n-1$

  3. 检验预测值

    1. 残差检验：计算相对残差
      $$
      \varepsilon(k)=\frac{x^{(0)}(k)-\hat{x}^{(0)}(k)}{x^{(0)}(k)},k=1,2,\cdots,n\
      $$
      若所有$|\varepsilon(k)|\lt0.1$，则认为达到较高要求，

      若所有$|\varepsilon(k)|\lt0.2$，则认为达到一般要求。

    2. 级比偏差值要求：
      $$
      \rho(k)=1-\frac{1-0.5a}{1+0.5a}\lambda(k),\lambda(k)为级比
      $$
      若所有$|\lambda(k)|\lt0.1$，则认为达到较高要求，

      若所有$|\lambda(k)|\lt0.2$，则认为达到一般要求。

灰色关联分析

灰色关联分析：分析系统中哪些因素是主要的，哪些是次要的，以及因素是正向还是逆向的。即因素间关联性、关联程度等。

灰色关联度计算步骤：

  1. 选取参考数列：即选择各参数最优情况构成理想样例$X_0={X_0(k)|k=1,2,\cdots,n}=(X_0(1),X_0(2),\cdots,X_0(m))$，m个比较数列（即需要计算与理想样例的关联度的样例）为$X_i={X_i(k)|k=1,2,\cdots,n}(i=1,2,\cdots,m)$其中$X_i(k)$表示第i个样例的第k个参数。（参数也可以表示时刻）

  2. 计算单个参数的关联度：
    $$
    \zeta_i(k)=\frac{min_imin_k|X_0(k)-X_i(k)|+\rho\cdot max_imax_k|X_0(k)-X_i(k)|}{|X_0(k)-X_i(k)|+\rho\cdot max_imax_k|X_0(k)-X_i(k)|}
    $$
    $\zeta_i(k)$表示第i个样本（比较数列）与参考数列在k时刻的关联系数，其中$\rho,0\le\rho\le1$为分辨系数，分辨系数越大，分辨率越大。

  3. 计算整体的关联度：
    $$
    r_i=\frac{1}{n}\sum^n_{k=1}\zeta_i(k)
    $$
    $r_i$表示第i个样本与参考数列的整体关联度。

  4. 判断正负关联：
    $$
    \sigma_i=\sum^n_{k=1}kX_i(k)-\sum^n_{k=1}X_i(k)\sum^n_{k=1}\frac{k}{n}\
    \sigma_n=\sum^n_{k=1}k^2-(\sum^n_{k=1}k)\frac{2}{n}\
    当sign(\frac{\sigma_i}{\sigma_n})=sign(\frac{\sigma_j}{\sigma_n}),则X_i和X_j为正关联\
    当sign(\frac{\sigma_i}{\sigma_n})=-sign(\frac{\sigma_j}{\sigma_n}),则X_i和X_j为负关联
    $$

插值

插值问题：已知函数在某区间内若干点处的值，求函数在该区间内其他点处的值。即找到一个函数（插值多项式）经过所有已知点。

分类：

  • 一维插值：二维平面。
  • 二维插值：
    • 网格插值：已知点步长相等，网格分布。
    • 散点插值：已知点散乱分布。

方法：

  • 线性插值：
    $$
    \frac{y-y_0}{x-x_0}=\frac{y_1-y_0}{x_1-x_0}
    $$
    代入x值即可求出y值。

  • 双线性插值：
    x与y分别进行一次线性插值。

  • 双立方（三次）插值：

    插值点(x, y)处的值通过最近的十六个采样点的加权平均得到，根据不同情况选择权重。

  • 拉格朗日插值法（容易出现Runge现象）：
    $$
    已知点(x_i,y_i),i=1,2,\cdots,k\
    则函数（插值多项式）为f(x)=\sum^k_{i=1}\prod_{j\ne i}\frac{x-x_j}{x_i-x_j}
    $$

  • 牛顿插值法：
    $$
    先定义:\

    二阶差商:f[x_i,x_j,x_k]=\frac{f[x_i,x_j]-f[x_j,x_k]}{x_i-x_k}(i\ne j)\
    \cdots\

    则:\
    f(x)=f(x_0)+fx,x_0\
    f[x,x_0]=f[x_0,x_1]+fx,x_0,x_1\
    \cdots\
    f[x,x_0,\cdots,x_{n-1}]=f[x_0,x_1,\cdots,x_n]+fx,x_0,x_1,\cdots,x_n\

回归分析

一元线性回归

一元线性回归模型：
$$
\begin{cases}
y=\beta_0+\beta_1x+\varepsilon\
E\varepsilon=0,D\varepsilon=\sigma^2
\end{cases}
$$
$\beta_0、\beta_1$称为回归系数，$x$称为回归变量，$Y=\beta_0+\beta_1x$称为$y$对$x$的回归直线方程。

一元线性回归分析的主要步骤：

  1. 用试验值（样本值）对$\beta_0、\beta_1和\sigma_1$作估计计算
  2. 对回归系数$\beta_0、\beta_1$作假设检验
  3. 利用得到的回归方程对y作预测和区间估计

最小二乘法

判断标准：二者之差的平方和最小。（求导找满足最小值的$\beta_0、\beta_1$）
$$
\begin{cases}
\hat{\beta}_1=\frac{n\sum Y_iX_i-\sum Y_i\sum X_i}{n\sum X_i^2-(\sum X_i)^2}\
\hat{\beta}_0=\bar{Y}-\hat{\beta}_1\bar{X}
\end{cases}
$$
离散形式：
$$
\
\begin{cases}
\bar{X}=\frac{1}{n}\sum X_i\
\bar{Y}=\frac{1}{n}\sum Y_i\
x_i = X_i-\bar{X}\
y_i = Y_i-\bar{Y}
\end{cases}\
\begin{cases}
\hat{\beta}_1=\frac{\sum x_iy_i}{\sum x_i^2}\
\hat{\beta}_0=\bar{Y}-\hat{\beta}_1\bar{X}
\end{cases}
$$
随机误差项的估计量：
$$
e_i=Y_i-\hat{Y}_i为第i个样本点的残差\
随机误差项方差的估计量:\hat{\delta}^2_e=\frac{\sum e_i^2}{n-2}
$$
回归方程的显著性检验：即对假设$H_0:\beta_1=0,H_1:\beta_1\ne 0$进行检验。

若假设$H_0$被拒绝，则回归显著，认为$y$和$x$存在线性关系，回归方程有意义。

否则回归不显著，不能用一元线性回归模型描述。

判断拒绝的方法：

  • F检验法：当$H_0$成立时，$F=\frac{U}{Q_e/(n-2)}\sim F(1,n-2)$

    其中$U=\sum^n_{i=1}(\hat{y}i-\bar{y})^2$（回归平方和），$Q_e=\sum^n{i=0}(y_i-\hat{y}_i)^2$

    则当$F\gt F_{1-\alpha}(1,n-2)$，拒绝$H_0$，否则接受

    $F(n_1,n_2)$的值查表

  • t检验法：当$H_0$成立时，$T=\frac{\sqrt{L_{xx}}\hat{\beta}_1}{\hat{\sigma}_e}\sim t(n-2)$

    其中$L_{xx}=\sum^n_{i=1}(x_i-\bar{x})^2=\sum^n_{i=1}x_i^2-n\bar{x}^2$

    则当$|T|\gt t_{1-\frac{\alpha}{2}}(n-2)$，拒绝$H_0$，否则接受

    $t(n)$的值查表

  • r检验法：$r=\frac{\sum^n_{i=1}(x_i-\bar{x})(y_i-\bar{y})}{\sqrt{\sum^n_{i=1}(x_i-\bar{x})^2}\sum^n_{i=1}(y_i-\bar{y})^2}$

    其中$r_{1-\alpha}=\sqrt{\frac{1}{1+(n-2)/F_{1-\alpha}(1,n-2)}}$

    则当$|r|\gt r_{1-\alpha}$时，拒绝$H_0$，否则接受

import statsmodels.formula.api as smf
import pandas as pd
 
def forward_selected(data, response):
    """
    使用Adjusted R-squared来评判新加的参数是否提高回归中的统计显著性
    Linear model designed by forward selection.
    Parameters:
    -----------
    data : pandas DataFrame with all possible predictors and response
    response: string, name of response column in data
    Returns:
    --------
    model: an "optimal" fitted statsmodels linear model
           with an intercept
           selected by forward selection
           evaluated by adjusted R-squared
    """
    remaining = set(data.columns)
    remaining.remove(response)
    selected = []
    current_score, best_new_score = 0.0, 0.0
    while remaining and current_score == best_new_score:
        scores_with_candidates = []
        for candidate in remaining:
            formula = "{} ~ {} + 1".format(response,
                                           ' + '.join(selected + [candidate]))
            score = smf.ols(formula, data).fit().rsquared_adj
            scores_with_candidates.append((score, candidate))
        scores_with_candidates.sort()
        best_new_score, best_candidate = scores_with_candidates.pop()
        if current_score < best_new_score:
            remaining.remove(best_candidate)
            selected.append(best_candidate)
            current_score = best_new_score
    formula = "{} ~ {} + 1".format(response,
                                   ' + '.join(selected))
    model = smf.ols(formula, data).fit()
 
    return model

综合评价问题分类：

  • 分类：对对象的全部个体进行分类
  • 比较、排序
  • 考察某一综合目标的整体实现程度：例如现代化的程度

构成综合评价问题的五个要素：被评价对象、评价指标、权重系数、综合评价模型和评价者。

综合评价的一般步骤：

  1. 确定综合评价的目的

  2. 确定评价指标体系

  3. 对指标数据做预处理

    1. 指标一致化：全部转化为效益型指标。

    2. 无量纲化处理：转化为可以相加的量纲。

      • 向量归一化：
        $$
        x=(x_1,x_2,\cdots,x_n)\
        归一化后:\
        x’=(\frac{x_1}{\sum^n_{i=1}x_i},\frac{x_2}{\sum^n_{i=1}x_i},\cdots,\frac{x_n}{\sum^n_{i=1}x_i})
        $$

      • 极差变换法（效益型）：结果均匀
        $$
        x_{ij}=\frac{a_{ij}-a_{min}}{a_{max}-a_{min}}
        $$

      • 线性比例变换法（效益型）：结果会右倾（趋向1）
        $$
        x_{ij}=\frac{a_{ij}}{a_{max}}
        $$

  4. 确定指标权重

  5. 求综合评价值

确定指标权重方法：

  • 主观定权法：
    • 专家评分法
    • 成对比较法
    • Santy权重法
  • 客观定权法
    • 模糊定权法
    • 秩和比法
    • 熵权法
    • 相关系数法

综合评价方法：

  • 线性加权综合法：要求各指标之间相互独立
    $$
    y=\sum^m_{j=1}w_jx_j
    $$
    权重系数对结果影响明显。

  • 非线性加权综合法：各指标间有较强关联性
    $$
    y=\prod^m_{j=1}x_j^{w_j}
    $$
    指数值只能为正数。

    容易拉开评价档次，对较小数值的变动敏感。

  • 理想点（TOPSIS）法：见上文

模糊综合评价

模糊综合评价：部分因素很难给出确切的表达，需使用模糊评价。
$$
设U是论域，称映射\
A(x):U\rightarrow [0,1]\
确定了一个U上的模糊子集A，映射A(x)称为A的隶属函数，\
表示x对A的隶属程度
$$
$A(x)=0.5$的点x称为A的过渡点，此点最具有模糊性。

当$A(x)$只取0或1时，模糊子集A就是经典子集。

一级模糊综合评价的基本步骤：

  1. 确定评价指标集U和评语集V

  2. 求出模糊评价矩阵P（一般根据具体数据），其中$P_{ij}$表示方案X的第i个指标处于第j级评语的隶属度。

    设各指标的权系数向量为：$A=(W_1,W_2,\cdots,W_n)$

  3. 计算综合模糊评价结果向量：$B=AP$，运算为模糊乘法：
    $$
    逻辑乘\and:两者取小\
    逻辑加\or:两者取大\
    eg.\
    a=(0.8,0.5,0.3)\
    b=(0.4,0.7,0.5)^T\
    ab=(0.8\and 0.4)\or(0.5\and 0.7)\or(0.3\and 0.5)
    $$

  4. 归一化评价结果向量后，计算综合评价结果：

    • 最大隶属度法：将最大隶属度对应等级作为评价等级
    • 加权平均法：给各等级赋权重，然后加权求和

多级模糊综合评价：

多级：即部分指标有子指标组成。eg.图像指标={3D指标，2D指标}，设子指标的比例为$A_{1\times 2}$，根据各子指标的模糊评价矩阵计算出评价结果向量$B_1,B_2$，则$A(B_1,B_2)^T$为父指标的模糊评价矩阵。

即从最小不可分的指标开始计算，看做一级模糊综合评价，逐步减少层次，最终计算。

变异系数法确定权重：

目的：使数值差异较大的指标能被明确区分，提高分辨能力，即权重更大。
$$
\bar{x}_i,s^2_i为第i项指标的均值与方差\
则第i项指标的权重为:w_i=v_i/\sum v_i,其中v_i=s_i/|\bar{x}_i|
$$

    • 主成分分析：侧重变量变换，变换时的因子数量等于原指标数。

相关性分析

基本变量：

  • 定类变量：根据定性区分，如性别、民族等
  • 定序变量：根据等级次序区分，如文化程度、规模等
  • 定距变量：根据大小或距离（有具体值）区分（无零点值），如温度、比率等
  • 定比变量：定距变量基础上增加了零点

两变量的相关分析

Pearson相关系数：即相关系数
$$
r=\frac{\sum^n_{i=1}(x_i-\bar{x})(y_i-\bar{y})}{\sqrt{\sum^n_{i=1}(x_i-\bar{x})^2\sum^n_{i=1}(y_i-\bar{y})^2}}\
检验其偶然性:\
t=\frac{r}{\sqrt{q-r^2/(n-2)}}\
|t|\gt t_{a/2}表明r统计显著\
|t|\le t_{a/2}表明r统计不显著,其中a为给定显著性水平
$$

适用条件：

  • 两变量均为连续变量
  • 两变量总体为正态分布或接近正态的单峰对称分布
  • 变量必须是成对的数据
  • 变量间为线性关系

Spearman等级相关系数：
$$
r_s=1-\frac{6\sum(x_i-y_i)^2}{n(n^2-1)},\其中x_i,y_i分别是两个变量按大小（或优劣）排序后对应的位次（值相同，对应位次相同）。\
当n\gt20时，需要进行t检验:\
t=r_s\sqrt{\frac{n-2}{1-r_s^2}}\
|t|\gt t_{a/2}表明r统计显著\
|t|\le t_{a/2}表明r统计不显著
$$
适用条件：

  • 两变量均为定序变量

$$
\chi^2 = \sum^n_{i=1}\frac{(o_i-e_i)^2}{e_i},\
其中o_i为实际值，e_i为期望值
$$

元胞自动机

元胞自动机（CA）：不可逆的时空离散的局部动力学模型，适合空间复杂系统的时空动态模拟研究。由一系列模型构造的规则构成（非严格的函数或方程）。

基本构成：元胞、元胞空间、邻居和规则。

一个元胞下一时刻的状态取决于本身和邻居的状态，邻居可以是距离大于1的元胞，需提前定义邻居。

核心：
$$
s_i^{t+1}=f(s_1^t,s_2^t,\cdots,s_m^t,s_i^t),其中s_1,s_2,\cdots,s_m为s_i的邻居元胞，f为转换规则。
$$

分类：

  • 平稳型：若干步后停留在一个固定的状态。
  • 周期型：若干步后在几种状态之间周期循环。
  • 混沌型：处于完全无序随机的状态。
  • 复杂型：演化过程中可能产生复杂的结构，该结构既不是完全的随机混乱，又没有固定的周期和状态。
