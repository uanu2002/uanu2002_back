Linux(2)--ssh使用

Posted by Uanu on Mon, Jan 1, 0001

ssh登录

基本用法

远程登录服务器：

1ssh user@hostname
  • user：用户名
  • hostname：IP地址或域名

第一次登录会提示：

1The authenticity of host '你的IP(你的IP)' can't be established.
2ECDSA key fingerprint is SHA256:iy237yysfCe013/l+kpDGfEG9xxHxm0dnxnAbJTPpG8.
3Are you sure you want to continue connecting (yes/no/[fingerprint])?

输入yes，然后回车。

这样服务器的信息就会记录在~/.ssh/known_hosts文件中。

最后输入密码即可。

默认登录端口号为22，如果需要登录特定端口：

1ssh user@hostname -p 22

配置文件

创建文件 ~/.ssh/config。

然后在文件中输入：

1Host myserver1
2    HostName IP地址或域名
3    User 用户名
4    port 端口（ssh默认22）
5
6Host myserver2
7    HostName IP地址或域名
8    User 用户名

之后再使用服务器时，可以直接使用ssh myserver1来登录服务器。

由于root用户权限过大，建议创建新用户并赋予sudo权限。

1adduser uanu # 创建用户uanu
2usermod -aG sudo uanu # 给uanu分配sudo权限

密钥登录

创建密钥：

1ssh-keygen

然后一直回车即可。

执行结束后，~/.ssh/目录下会多两个文件：

  • id_rsa：私钥
  • id_rsa.pub：公钥 之后想免密码登录哪个服务器，就将公钥传给哪个服务器即可。

例如，想免密登录myserver服务器。则将公钥中的内容，复制到服务器中的~/.ssh/authorized_keys文件里即可。

也可以使用如下命令一键添加公钥：

1ssh-copy-id myserver

执行命令

格式：

1ssh user@hostname command

示例：

1# 单引号中的$i可以求值
2ssh myserver 'for ((i = 0; i < 10; i ++ )) do echo $i; done'

scp传文件

基本用法

格式：

1scp source destination # 将source路径下的文件复制到destination中
2scp source1 source2 destination # 一次复制多个文件
3scp -r ~/tmp myserver:/home/Uanu/ # 复制文件夹

指定服务器的端口号：

1scp -P 22 source1 source2 destination

注：scp的-r -p等参数尽量加在source和destination之前。

使用scp配置其他服务器的vim和tmux：

1scp ~/.vimrc ~/.tmux.conf myserver:

安装软件

1sudo apt-get updates
2sudo apt-get install tmux
