泛型

JDK5引入，泛型本质是参数化类型，即将数据类型看做参数。

泛型方法

通过传入不同参数类型，来实现对不同类型数据的处理，类似于重载。

规则：

  • 声明时，在方法返回类型前有一个类型参数声明部分，以尖括号<>分隔。
  • 类型参数声明部分包括一个或多个类型参数（用泛型标记符表示），逗号隔开。
  • 类型参数可以用来声明返回值类型，也可以作为传入参数类型的占位符。
  • 类型参数只能代表引用型类型，不能是原始类型，如：int、double等。

泛型标记符

  • E - Element (在集合中使用，因为集合中存放的是元素)
  • T - Type（Java 类）
  • K - Key（键）
  • V - Value（值）
  • N - Number（数值类型）
  • ？ - 表示不确定的 java 类型
 1public class demo{
 2    public static <E> E is_print(E[] Array){
 3        for(E element : Array){
 4            System.out.printf("%s", element);
 5        }
 6        E re = Array[0];
 7        return re;
 8    }
 9    public static void main(String [] argv){
10        Integer[] array = { 1, 2, 3, 4, 5 };
11        int re = is_print(array);
12        System.out.println(re);
13    }
14}
15// 输出：123451

在类名后面添加类型参数声明部分，用法同泛型方法。

 2   
 3  private T t;
 4

 6    this.t = t;
 7  }
 8

11  }
12 
13  public static void main(String[] args) {
14    Box<Integer> integerBox = new Box<Integer>();
