参数裁剪和量化

量化和二进制化

参数量化：将实数转换成一个离散整数且保证计算精度的情况下使用最小比特数。

$Q(r)=Int(r/S)-Z$，$Q$是量化运算符，$r$是实数值输入，$S$是实数值的比例因子，$Z$是整数零点，$Int$函数将实数四舍五入为整数。

量化运算后间隔均匀称为均匀量化，否则为非均匀量化。

$S=\frac{\beta - \alpha}{2^{b}-1}$，$[\alpha,\beta]$是实数值$r$的上下边界，$b$是量化比特位宽度。

上下边界绝对值相等称为对称量化，否则为非对称量化。边界由权重静态计算成为静态量化，运行时动态运算为动态量化。

网络修剪（剪枝）

  • 非结构化剪枝：对于独立权重（individual weight）进行剪枝。

    缺点：得到的权重矩阵是稀疏的，需要专用的硬件和库。

  • 结构化剪枝：对通道或层进行剪枝，原始卷积结构仍存在。

典型的修剪算法：训练->剪枝->微调

常用剪枝算法：

  • 权重共享：即使用卷积，卷积的权重对于一张图的每个部分都相同，相比全连接层减少了参数量。

  • 卷积核剪枝：删除权重绝对值之和最小的卷积核及其feature map和相关核。

  • 通道剪枝：如果某一层输入子集得到的输出与原来很类似，就可以去掉该子集以外的输入和对应的filter。
