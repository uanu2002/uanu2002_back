第三题

强烈建议无论简单复杂的状态机都分为三部分：转移逻辑、输出逻辑、状态保存。

转移逻辑和输出逻辑单独写个电路组件，方便修改时重新生成组合电路。

虽然这次没考到，但有限状态机的初值（例如斐波那契）设置方式（用计数器和多路选择器），电路的同步复位（增加寄存器或多路选择器）也是常见考点。

另外，可以掌握一些好用的冷门的组件，说不定哪天就用上）。

冷门组件

Pull Register

将浮点值变为0/1/error

  • Pull Direction：目标值

Bit Extender

不同位宽转换，代替分线器

多->少：截断保留低位部分

少->多：低位相同，高位补充0/1

  • Extension Type：高位补充值
    • 0：全部补0
    • 1：全部补1
    • Sign：全部补输入的最高位、
    • Input：额外增加输入，全部补输入(0/1/x)

ODD Parity

输入有奇数个1，输出1，否则为0，也可以用float。

Even Parity

输入有偶数个1，输出1，否则为0，也可以用float。

Bit Adder

输出所有输入中1的数量，或所有输入各位相加的和。

Bit Finder

输出输入中第一个/最后一个0/1的索引。

  • Type：寻找类型
    • Lowest-order 1：第一个1
    • Highest-order 1：最后一个1
    • Lowest-order 0：第一个0
    • Highest-order 0：最后一个0
