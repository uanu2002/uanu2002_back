Windows安装vscode配置C/C++

1、下载vscode

2、下载gcc编译器

官网下载：MinGW-w64 - for 32 and 64 bit Windows - Browse Files at SourceForge.net

选择最新版本MinGW-W64 GCC，根据电脑选择对应压缩包下载（64位windows电脑下载x86_64-win32-seh）。

下载后解压，找到bin文件夹，复制bin的路径，添加到电脑的环境变量Path中。

然后在命令行cmd中输入

1gcc -v

如果显示gcc位置、版本等许多信息则gcc安装成功。

3、设置配置文件

（1）在vscode中打开项目文件（下面设置的编译器等配置只在这个文件内有效）。

（2）配置编译器文件 Ctrl+Shift+P打开命令面板，输入C/c++:Edit Configurations(UI)，点击打开。

如下图所示，

1：windows系统选择win32 2：填写：bin路径\g++.exe 3：可不填，为编译器参数 4：选择gcc-x64

设置好后文件中会出现.vscode文件夹，其中只有一个c_cpp_properties.json的配置文件，如果出现其他文件则删除。

（3）配置构建文件 Ctrl+Shift+P打开控制面板，输入Tasks:Configure Default Build Task，点击后再输入C/C++: g++.exe build active file。然后会在.vscode文件夹内生成一个tasks.json配置文件。

（4）配置调试文件

新建或打开一个C++文件，然后点击菜单栏中的调试C/C++文件或Debug，然后选择C++(GDB/LLDB)，然后会在.vscode文件夹内生成一个launch.json配置文件。

执行完效果如下图。

配置好后可以单独保存.vscode文件夹，下次配置时直接复制到项目文件夹内即可。

此处粘贴个人的三个文件夹内容，如果上述操作无法完成，则可以直接创建.vscode文件夹和三个配置文件，然后把内容粘贴，然后要修改其中的编译器等地址和版本信息！

Mac安装vscode配置C/C++

一、安装vscode

1、下载安装vscode

mac版本的vscode下载链接

下载后直接安装即可。

2、vscode基础设置

主题选择

刚进入vscode时会弹出主题的选择，四个主题任选其一即可，然后点击左上角关闭Get Started页面。

插件安装

点击左侧导航栏扩展按钮，进入插件商店。

搜索栏输入chinese，点击打开第一个标有中文简体的插件，然后右侧点击install下载，下载好则会显示为uninstall。

下载好后，会有弹窗提示是否重启vscode，选择Restart。如果没有弹窗提示则手动关闭vscode再打开。重新打开后，界面变为中文则插件生效。

下面用同样的方法安装C/C++，在扩展中搜索C/C++，选择安装第一个插件。

二、C语言环境配置

1、创建C文件

点击打开左侧导航栏资源管理器。

然后点击打开文件夹。

新建或选择一个文件夹，这个文件夹用于存放以后的源文件和程序。

点击打开后会有弹窗提示，先选择好，然后勾选信任父文件夹"xxx"中所有文件的作者，最后点击是，我信任此作者。

点击新建文件夹，输入demo.c并回车。demo是文件名可以任意修改。

2、生成配置文件

点击demo.c打开文件，点击右上角的向下尖括号。

选择运行C/C++文件。

然后会弹出编译器选择，选择C/C++: clang生成和调试活动文件。

然后左侧会文件夹内会多出一个.vscode文件夹，里面包含两个配置文件。

3、程序测试

再次打开demo.c，输入以下内容。

1#include <stdio.h>
2
3int main(){
4    printf("hello world");
5    return 0;
6}

然后再次点击运行C/C++文件。

如果有弹窗则先记住选择，然后选择仍要调试。

运行后右下侧会弹出终端，选择调试控制台。

如果看到hello world则表示配置成功。

三、注

  • .vscode不要删除。
