Linux(1)--shell语法

Linux 操作系统
操作系统
Publish Date:   2022-07-18

简介

shell是我们通过命令行与操作系统沟通的语言。shell脚本可以直接在命令行中执行，同时方便复用。
因此，终端中的命令行可以看成是一个“shell脚本在逐行执行”。Linux中常见的shell脚本有很多种，常见的有：

Bourne Shell(/usr/bin/sh``或``/bin/sh)
Bourne Again Shell(/bin/bash)
C Shell(/usr/bin/csh)
K Shell(/usr/bin/ksh)
…

在编写脚本时需要制定脚本解释器，例如bash脚本需要再文件开头写#! /bin/bash。

运行方式

作为可执行文件

:~$ chmod +x test.sh  #chmod +x使得脚本具有可执行权限
:~$ ./test.sh #在当前路径下执行脚本
:~$ /home/uanu/test.sh #在绝对路径下执行脚本

使用解释器执行（不需要执行权限）

:~$ bash test.sh #在当前目录下用解释器执行脚本

注释

单行注释

# 单行注释
echo "Uanu's blog"

多行注释

:<<str
第一行注释
第二行注释
第三行注释
str

str可以为任意字符串

变量

定义变量

无需声明类型，类似python。变量本质都是字符串。

var1='Uanu' # 单引号定义字符串
var2="Uanu" # 双引号定义字符串
var3=Uanu   # 不加引号也可以定义字符串

定义只读变量

变量前加上readonly或者declare

var=Uanu
readonly name # 定义只读变量
declare -r name # 定义只读变量

var=blog # 此时会报错，因为var为只读变量

使用变量

使用变量需要在变量名前加上$符号或者${},$可以看做获取变量的值，其中{}是可选的，用于帮助解释器识别变量边界。

var="Uanu" # 定义变量
echo $name # 输出：Uanu
echo $name blog # 输出：Uanu blog

删除变量

使用unset删除变量

var=Uanu
unset var # 删除变量var
echo $var # 输出：空行
# ps：未定义变量输出时为空行

变量类型

自定义变量（局部变量） 环境变量（全局变量）
子进程不能访问 子进程可以访问

类型转换

自定义变量转环境变量

var=Uanu
# 第一种方法
export var
# 第二种方法
declare -x var

环境变量转自定义变量

export var=Uanu # 定义环境变量
declare +x var # 转自定义变量

默认变量

执行shell脚本时，可以向脚本传递参数，$1是第一个参数，$2是第二个参数，以此类推。**$0是文件名**（包含路径）。

其他变量

参数 说明
$# 代表文件传入的参数个数
$* 由所有参数构成的用空格隔开的字符串，如"$1 $2"
$@ 每个参数分别用双引号括起来的字符串，如"$1" "$2"
$$ 脚本当前运行的进程ID
$? 上一条命令的退出状态（注意不是stdout，而是exit code）。0表示正常退出，其他值表示错误
$(command) 返回command的stdout（可嵌套）
`command` 返回command的stdout（不可嵌套）

注

单引号与双引号区别

单引号中内容会原样输出，不会执行其中的语句，不会获得变量的值。

双引号中语句会执行，会获得变量的值。

var=Uanu
echo '$var blog' #输出：$var blog
echo "$var blog" #输出：Uanu blog

获取字符串长度

>var=Uanu
>echo ${#name} # 输出：3

#常用于长度相关

提取子串

>var="Uanu's blog"
>echo ${var:0:5} #输出：Uanu'

数组

区别于C语言的数组，shell语法中的数组可以存放多个不同类型的值，且支持一维数组，初始化不需要指明数组大小。下标从0开始。

数组变量定义

数组用小括号表示，元素之前用空格隔开。

array=(1 abc "Uanu" xxx)

可以定义单独定义数组中某个元素的值

array[0]=1
array[1]=abc
array[10000]="Uanu"

数组实际的大小并不由下表决定，而是由定义的元素个数决定，下标只作为索引使用。如上定义的数组大小其实为3。

数组读取

元素读取

${array[index]}

整个数组读取

${array[@]} # 第一种方式
${array[*]} # 第二种方式

长度读取

${#array[@]} # 第一种方式
${#array[*]} # 第二种方式

expr命令

expr命令用于表达式求值

expr 表达式

表达式说明

  1. 要用空格隔开每一项
  2. 需要用反斜杠\放在特定字符前
  3. 对包含空格和其他特殊字符的字符串要用引号括起来
  4. expr会在stdout中输出结果。若为逻辑关系表达式，若结果为真，stdout为1，否则为0
  5. expr的exit code：若为计算则为真。若为逻辑关系表达式则取决于逻辑关系表达式真假。注意：**exit code真时为0，假时为1。**

stdout可以看做C语言中的输出，exit code可以看做C语言中的返回值return。

表达式分类

字符串表达式

  • length str

    返回str的长度

  • index str charset

    charset中任意单个字符在str中最前面的字符位置，下标从1开始。如果str中不存在charset中的字符，则返回0。

  • substr str position length

    返回str从position开始，长度最大为length的子串。如果position或length为负数、0或非数值，则返回空字符串。

示例：

str="Hello World!"

echo `expr length "$str"`  # ``不是单引号，表示执行该命令，输出12
echo `expr index "$str" aWd`  # 输出7，下标从1开始
echo `expr substr "$str" 2 3`  # 输出 ell

整数表达式

算法表达式优先级低于字符串表达式，高于逻辑关系表达式。

  • + -

    加减运算，两端参数会转换为整数，如果失败则报错。

  • * / %

    乘、除、取模运算。两端参数会转换为整数，如果转换失败则报错。

  • ()

    可以表示优先级，但需要反斜杠转义。

示例：

a=3
b=4

echo `expr $a + $b`  # 输出7
echo `expr $a - $b`  # 输出-1
echo `expr $a \* $b`  # 输出12，*需要转义
echo `expr $a / $b`  # 输出0，整除
echo `expr $a % $b` # 输出3
echo `expr \( $a + 1 \) \* \( $b + 1 \)`  # 输出20，值为(a + 1) * (b + 1)

逻辑关系表达式

  • |

    如果第一个参数非空且非0，则返回第一个参数的值，否则返回第二个参数的值，但要求第二个参数的值也是非空或非0，否则返回0。如果第一个参数是非空或非0时，不会计算第二个参数。

  • &

    如果两个参数都非空且非0，则返回第一个参数，否则返回0。如果第一个参为0或为空，则不会计算第二个参数。

  • < <= = == != >= >

    比较两端的参数，如果为true，则返回1，否则返回0。”==”是”=”的同义词。”expr”首先尝试将两端参数转换为整数，并做算术比较，如果转换失败，则按字符集排序规则做字符比较。

  • ()

    可以表示优先级，但需要反斜杠转义。

示例：

a=3
b=4

echo `expr $a \> $b`  # 输出0，>需要转义
echo `expr $a '<' $b`  # 输出1，也可以将特殊字符用引号引起来
echo `expr $a '>=' $b`  # 输出0
echo `expr $a \<\= $b`  # 输出1

c=0
d=5

echo `expr $c \& $d`  # 输出0
echo `expr $a \& $b`  # 输出3
echo `expr $c \| $d`  # 输出5
echo `expr $a \| $b`  # 输出3

read命令

read命令用于从标准输入中读取单行数据，当读到文件结束符ctrl+d时，exit code为1，否则为0。

参数说明：

  • -p：后面可以接提示信息。
  • -t：后面跟秒数，定义输入字符的等待时间，超过等待时间后会自动忽略此命令，执行后续命令。

示例：

read name  # 读入name的值
Uanu  # 标准输入
echo $name  # 输出name的值
Uanu  #标准输出
read -p "Please input your name: " -t 30 name  # 读入name的值，等待时间30秒
Please input your name: Uanu  # 标准输入
echo $name  # 输出name的值
Uanu  # 标准输出。

echo命令

echo命令用于输出字符串。

echo str

显示普通字符串

echo “Uanu”
ehco Uanu # 字符串引号可以忽略

显示转义字符

echo "\"Uanu blog\""  # 注意只能使用双引号，如果使用单引号，则不转义
echo \"Uanu blog\"  # 也可以省略双引号

显示变量

var=Uanu
echo "$var"  # 输出：Uanu

显示换行

echo -e "Hi\n"  # -e 开启转义
echo "Uanu"
# 输出：
Hi

Uanu

显示不换行

echo -e "Hi \c" # -e 开启转义 \c 不换行
echo "Uanu"
# 输出：
Hi Uanu

原样输出字符串

var=Uanu
echo '$var\"'
# 输出：$var\

显示命令的执行结果

echo `date`
# date命令返回当前日期和时间
# 输出：xxx-xxx-xx-xx:xx:xx XXX xxxx

printf命令

printf命令基本上与C语言中用法相同。

与echo区别：

echo字符串末尾默认添加换行符

printf不会在字符串末尾添加换行符

printf format-string [arguments...]

示例：

printf "%10d.\n" 123  # 占10位，右对齐
printf "%-10.2f.\n" 123.123321  # 占10位，保留2位小数，左对齐
printf "My name is %s\n" "yxc"  # 格式化输出字符串
printf "%d * %d = %d\n"  2 3 `expr 2 \* 3` # 表达式的值作为参数
# 输出：
       123.
123.12    .
My name is yxc
2 * 3 = 6

判断

逻辑运算符&&和||

  • &&表示与，||表示或

  • 二者都具有短路原则

    expr1 && expr2：当expr1为假时，直接忽略expr2

    expr1 || expr2：当expr1为真时，直接忽略expr2

逻辑运算符很多时候可以起到判断语句的作用。

test命令

ps：可用查询命令用法

man command

test命令用于判断文件类型和变量比较

test命令用exit code返回结果。

示例：

test 2 -lt 3  # 为真，返回值为0
echo $?  # 输出上个命令的返回值，输出0
test -e test.sh && echo "exist" || echo "Not exist"
exist  # test.sh 文件存在
test -e test2.sh && echo "exist" || echo "Not exist"
Not exist  # testh2.sh 文件不存在
类型判断参数 代表意义
-e 文件是否存在
-f 是否为文件
-d 是否为目录
权限判断参数 代表意义
-r 文件是否可读
-w 文件是否可写
-x 文件是否可执行
-s 是否为空文件

格式：

test -参数 filename
整数比较参数 代表意义
-eq （前者）是否等于（后者）
-ne 是否不等于
-gt 是否大于
-lt 是否小于
-ge 是否大于等于
-le 是否小于等于

格式：

test $a -eq $b  # a是否等于b
字符串比较参数 代表意义
test -z STRING 判断STRING是否为空，如果为空，则返回true
test -n STRING 判断STRING是否非空，如果非空，则返回true（-n可以省略）
test str1 == str2 判断str1是否等于str2
test str1 != str2 判断str1是否不等于str2
多重判断参数 代表意义
-a 是否同时成立
-o 是否至少一个成立
！ 取反

格式：

test -r filename -a -x filename

判断符号[]

[]与test用法几乎一样，常用于if语句中，另外[[]]是[]的加强版，支持更多特性。

示例：

[ 2 -lt 3 ]  # 为真，返回值为0
echo $?  # 输出上个命令的返回值，输出0
[ -e test.sh ] && echo "exist" || echo "Not exist"
exist  # test.sh 文件存在
[ -e test2.sh ] && echo "exist" || echo "Not exist"
Not exist  # testh2.sh 文件不存在

注：

  • []每一项都要用空格隔开
  • 变量最好用双引号括起来
  • 常数最好用单或双引号括起来

示例：

name="Uanu blog"
[ $name == "Uanu blog" ]  # 错误，等价于 [ Uanu blog == "Uanu blog" ]，会报错：参数太多
[ "$name" == "Uanu blog" ]  # 正确

判断语句

单层if

if condition
then
    语句1
    语句2
    ...
fi

示例：

a=3
b=4

if [ "$a" -lt "$b" ] && [ "$a" -gt 2 ]
then
    echo ${a}在范围内
fi

单层if-else

if condition
then
    语句1
    语句2
    ...
else
    语句1
    语句2
    ...
fi

示例：

a=3
b=4

if ! [ "$a" -lt "$b" ]
then
    echo ${a}不小于${b}
else
    echo ${a}小于${b}
fi

多层if-elif-elif-else

if condition
then
    语句1
    语句2
    ...
elif condition
then
    语句1
    语句2
    ...
elif condition
then
    语句1
    语句2
else
    语句1
    语句2
    ...
fi

示例：

a=4

if [ $a -eq 1 ]
then
    echo ${a}等于1
elif [ $a -eq 2 ]
then
    echo ${a}等于2
elif [ $a -eq 3 ]
then
    echo ${a}等于3
else
    echo 其他
fi

case…ease形式

case $变量名称 in
    值1)
        语句1
        语句2
        ...
        ;;  # 类似于C/C++中的break
    值2)
        语句1
        语句2
        ...
        ;;
    *)  # 类似于C/C++中的default
        语句1
        语句2
        ...
        ;;
esac

示例：

a=4

case $a in
    1)
        echo ${a}等于1
        ;;  
    2)
        echo ${a}等于2
        ;;  
    3)                                                
        echo ${a}等于3
        ;;  
    *) # 表示其他情况。类似C语言的default
        echo 其他
        ;;  
esac

循环语句

for…in…do…done

for var in val1 val2 val3
do
    语句1
    语句2
    ...
done

示例：

for i in a 2 cc
do
    echo $i
done

for file in `ls` # 即把ls返回的结果作为参数
do
    echo $file
done

for i in $(seq 1 10) # $(seq 1 10)会返回1-10的序列
do
    echo $i
done

for i in {a..z} # 返回a到z的序列，数字也可，倒序也可。
do
    echo $i
done

for ((…;…;…)) do…done

类似C语言的for循环

for ((expression; condition; expression))
do
    语句1
    语句2
done

示例：

for ((i=1; i<=10; i++))
do
    echo $i
done

while…do…done循环

为假时结束循环。

while condition
do
    语句1
    语句2
    ...
done

示例：

while read name #当读到文件结束符``ctrl+d``，返回0

为真时结束循环。

until condition
do
    语句1
    语句2
    ...

示例：

    read -p "Please input yes/YES to stop this program: " word

函数

类似C语言中的函数，返回的exit code默认是0,0表示正常结束，其余均为不正常。获取输出结果和返回值方法参照变量中的其他变量部分。

定义函数

[function] func_name() {  # function关键字可以省略
    语句1
    语句2
    ...
}

函数参数

$1表示第一个输入参数，依次类推，但$0仍然为文件名（同上）。

示例：

func() {  # 递归计算 $1 + ($1 - 1) + ($1 - 2) + ... + 0
    word=""
    while [ "${word}" != 'y' ] && [ "${word}" != 'n' ]
    do
        read -p "要进入func($1)函数吗？请输入y/n：" word
    done

    if [ "$word" == 'n' ]
    then
        echo 0
        return 0
    fi  

    if [ $1 -le 0 ] 
    then
        echo 0
        return 0
    fi  

    sum=$(func $(expr $1 - 1))
    echo $(expr $sum + $1)
}

echo $(func 10) # 传入参数10
# 输出：55

局部变量

可以在函数内部定义局部变量。

local 变量名=变量值

exit命令

exit命令用于退出当前的shell进程，并返回一个退出状态exit code。

示例：

#! /bin/bash

if [ $# -ne 1 ]  # 如果传入参数个数等于1，则正常退出；否则非正常退出。
then
    echo "arguments not valid"
    exit 1 # 不正常返回
else
    echo "arguments valid"
    exit 0 # 正常返回
fi

文件重定向

每个进程都会默认打开3个文件描述符：

  • stdin标准输入，从命令行读取数据，文件描述符为0。
  • stdout标准输出，向命令行输出数据，文件描述符为1。
  • stderr标准错误输出，向命令行输出数据，文件描述符为2。

可以通过文件重定向将这三个文件重定向到其他文件中。

命令 说明
command > file 将stdout重定向到file中
command < file 将stdin重定向到file中
command >> file 将stdout以追加方式重定向到file中
command n> file 将文件描述符n重定向到file中
command n>> file 将文件描述符n以追加方式重定向到file中

示例：

echo -e "Hello \c" > output.txt  # 将stdout重定向到output.txt中
echo "World" >> output.txt  # 将字符串追加到output.txt中

read str < output.txt  # 从output.txt中读取字符串

echo $str  # 输出结果：Hello World

同时重定向stdin和stdout

./test.sh < input.txt > output.txt

引入外部脚本

类似C语言的#include <filename>

格式：

. filename  # 注意点和文件名之间有一个空格
或
source filename

示例：

#! /bin/bash

source test1.sh # 或 . test1.sh

echo My name is: $name  # 可以使用test1.sh中的变量
