基础知识

Java特点

  • 面向对象：将需求分类实现。（面向过程：将需求分分布实现）
  • 安全性高：不支持指针类型、自动垃圾回收、字节码验证、异常处理机制。
  • 结构特点：Java项目的结构为：项目 -> 包 -> 源代码。每个项目至少包含一个类，类中包含方法。
  • …

Java执行过程

.java文件 -> .class字节码文件 -> 运行

// 代码运行main.java
// step 1
javac main.java // javac.exe进行编译并生成main.class字节码文件
// step 2
java main // java.exe翻译执行main.class

Java安装

1、下载安装JDK

JDK是Java开发工具包，即提供上述需要的javac.exe、java.exe等运行环境和各种类库和工具。

JDK官网下载

2、配置环境变量

右键此电脑选择属性 -> 高级系统设置 -> 高级 -> 环境变量 -> 在系统变量（用户变量也行）进行以下操作：

  • 设置JAVA_HOME的值为JDK的安装目录。
  • PATH添加java.exe和javac.exe的位置。
  • 设置CLASSPATH的值为类库的路径。

此时就可以用记事本直接编写Java程序，然后更改后缀为,java，再使用上述执行方法即可。

（可选）3、下载安装IntelliJ idea

Intellij idea类似vscode，提供集成开发环境，可以更方便的开发和管理。

Intellij官网下载

下载后直接新建Java项目即可。

Java概念

方法：类比C语言中的函数。

实例变量：类比C语言中的变量。

类：提供一个模板，里面包含实例变量、方法等。eg：人类：有头发，会说话。

对象：类具化的结果。eg：人类的小明：有黑头发，会说北京话。

命名：

  • 大小写敏感。
  • 类名首字母应该大写。
  • 方法名小写开头。
  • 源文件名必须与类名相同。
  • 程序都必须由public static void main(String[] args)方法开始执行。

变量：

  • 局部变量：同C语言。
  • 类变量：同C语言区局变量。

注释

单行、多行注释：同C语言。
文档注释：即作者、来源、参数等关于文档的信息。使用/** */。

// 示例main.java
/**
* @author uanu //@author、@version均为标签，后面接描述。
* @version 1.0.0
*/
// 代码部分

可以通过javadoc [options] [sourcefiles]命令获得文档注释。

javadoc -author main.java // 获得author的文档描述：uanu
